System Structure
Actions:
  • Show complete structure with all nodes and edges.
  • Switch to interactive mode to focus on a node and its neighbours.
Semantics:
  • A rectangular box represents a microservice.
  • A cylinder represents a RabbitMQ message exchange.
  • A black link represents an asynchronous information flow from source to target. If both source and target are connected to a microservice then a message exchange exists in between them. This message exchange is not shown.
  • A red link represents a synchronous information flow (note: the actual synchronous call takes place in the opposite direction).
